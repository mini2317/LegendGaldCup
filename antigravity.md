# LegendGaldCup

## 소개
프로젝트명 : 레전드 갈드컵
디스코드 봇을 활용한 설문조사 및 의견 받음

## 세부사항
1. 설문조사
- 봇은 gemini api를 활용하여 주제를 선정하는 '마스터'역할을 함
- 설문조사 기간은 3일
- 설문조사가 끝나면 전체 의견들의 선택 비율과 무작위 서버 하나의 반응을 다른 무작위 서버로 익명으로 전달함
- 전달이 끝나면 다음 설문조사가 시작됨
- 전체 서버들의 반응도 원한다면 볼 수 있음. 이 경우, 서버 id로 조회해야함.

2. 의견 받기
- 설문조사에 대해 의견을 적으면서 투표를 할 수 있음
- 정보는 익명으로 수집됨. DB상에서는 유저 id를 저장해서 중복투표를 방지하지만, 의견은 익명으로 처리되고 서버id만 저장같이 저장되어 누구의 의견인지 알 수 없음.
- 의견은 300자 이내로 제한
- 의견을 제시하는 것은 설문조사 선택 옵션 중 하나를 선택하고, 의견을 덧붙이는 방식으로 진행함

3. 설문조사 주제 제시
- 아무때나 설문조사 주제를 제시할 수 있음
- 제시된 주제는 진행중인 설문조사가 끝나면 무작위로 제시된 주제 하나를 다음 설문조사 주제로 채택하는데, 이때 gemini api로 적절한 주제인지 판단함
- 적절한 주제가 하나도 없다면, 마스터가 직접 주제를 제시함 (이 경우 마스터가 주제를 제시했다는 사실을 알림)

## 주의사항
- 한국인이 대상이므로 명령어도 한국어로 진행함
- 봇의 이름은 레전드 갈드컵
- 명령어들에 맞춰서 도움말들도 충분히 작성할 것
- 소개 명령어를 만들어서 봇에 대한 소개를 할 수 있도록 할 것
- 나와 대화할 때는 대화를 한국어로 진행할 것
- .env에 gemini api key와 discord bot token을 저장하는 것을 잊지 말 것
